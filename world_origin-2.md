# Система точек отсчёта (World Origin System)

---

## Концепция: два маркера

В системе существуют **две разные точки отсчёта**:

### 1. Глобальный (0,0) — абсолютный ноль
- Существует всегда, **никуда не двигается**
- Это абсолютный ноль всей сцены, как начало координат на математической плоскости
- Визуально: **серый** крестик, ненавязчивый
- Нельзя перетащить мышью — только для ориентира

```
      │
──────+──────   серый, неподвижный
      │
  глобальный (0,0)
```

### 2. Кастомный origin — пользовательская точка отсчёта
- Опциональный маркер, который пользователь может поставить **в любое место**
- Все новые фигуры создаются с координатами **относительно него**
- Все координаты в панели свойств отображаются относительно него (пока он активен)
- Визуально: **красный** крестик с кружком, перетаскиваемый
- При сбросе — исчезает, координаты пересчитываются обратно относительно глобального (0,0)

```
      │
──────⊕──────   красный, перетаскиваемый
      │
  кастомный origin
```

---

## Создание кастомного origin

Кастомный маркер изначально отсутствует. Способ создать — двойной клик на пустом месте холста. Маркер появляется в точке клика.

```
Двойной клик на пустом месте холста:
  if кастомный origin не существует:
    создать маркер в точке клика
    пересчитать координаты всех фигур относительно нового маркера
  else:
    переместить существующий маркер в точку клика
    пересчитать координаты всех фигур
```

---

## Перемещение кастомного origin мышью

```
Hover над красным маркером:
  курсор → sf::Cursor::Hand

MouseButtonPressed (Left) на маркере:
  state = DRAGGING_CUSTOM_ORIGIN

MouseMoved (while DRAGGING_CUSTOM_ORIGIN):
  customOriginPos = screenToWorld(mousePos)
  пересчитать координаты всех фигур в реальном времени

MouseButtonReleased:
  state = IDLE
```

**Фигуры визуально стоят на месте.** Меняются только числа в панели свойств.

---

## Пересчёт координат фигур

```cpp
void Scene::setCustomOrigin(sf::Vector2f newOriginWorld) {
    sf::Vector2f delta = newOriginWorld - customOriginPos;
    for (auto& figure : figures) {
        figure->anchor -= delta;
        // Вершины (относительные координаты) не трогаем
    }
    customOriginPos = newOriginWorld;
}
```

Координаты в панели свойств = `figure->anchor` + относительные координаты вершин, всё в пространстве активного origin.

---

## Сброс кастомного origin

При сбросе:
- Кастомный маркер **исчезает**
- Координаты всех фигур пересчитываются обратно относительно **глобального (0,0)**
- Фигуры визуально не двигаются

```cpp
void Scene::resetCustomOrigin() {
    if (!customOriginActive) return;
    // Пересчитать обратно относительно глобального (0,0)
    sf::Vector2f delta = customOriginPos - globalOriginPos; // globalOriginPos = {0,0}
    for (auto& figure : figures) {
        figure->anchor += delta;
    }
    customOriginActive = false;
}
```

Способы сброса:

| Способ | Действие |
|---|---|
| Двойной клик по красному маркеру | Сброс — маркер исчезает |
| Кнопка `[Сбросить]` в панели свойств | То же самое |

---

## В панели свойств

```
┌──────────────────────────────────┐
│  ТОЧКА ОТСЧЁТА                  │
│  ● Глобальная (0, 0)            │  ← всегда, неизменна
│                                  │
│  ○ Кастомная                    │  ← если активна
│    X: [ 214.0 ]  Y: [ -87.5 ]  │
│                    [Сбросить]    │
└──────────────────────────────────┘
```

- Если кастомный origin не создан — показывается только глобальная
- Кнопка `[Сбросить]` появляется только когда кастомный origin активен

---

## Какие координаты показывать в панели свойств фигур

- Если кастомный origin **активен** → все координаты фигур относительно него
- Если кастомный origin **не активен** → все координаты относительно глобального (0,0)

---

## Взаимодействие с другими системами

| Система | Поведение |
|---|---|
| Зум | Оба маркера масштабируются вместе с холстом |
| Pan (Space + ЛКМ) | Оба маркера двигаются вместе с холстом — они часть мирового пространства |
| Создание новой фигуры | Anchor новой фигуры задаётся в координатах активного origin |
| Bounding box | Пересчитывается автоматически |
| Оси X/Y (toggle `A`) | Рисуются от активного origin |

## Чеклист

### Глобальный (0,0)
- [ ] Серый крестик всегда виден на холсте
- [ ] Нельзя перетащить мышью
- [ ] Служит точкой отсчёта когда кастомный origin не активен

### Кастомный origin
- [ ] По умолчанию отсутствует
- [ ] Двойной клик на пустом холсте — создать / переместить маркер
- [ ] Красный крестик с кружком (⊕), перетаскиваемый мышью
- [ ] Hover — курсор меняется на руку
- [ ] Drag — маркер двигается, фигуры стоят на месте
- [ ] Координаты всех фигур пересчитываются в реальном времени при drag
- [ ] Двойной клик по маркеру — сброс (маркер исчезает)
- [ ] Кнопка `[Сбросить]` в панели свойств — то же самое
- [ ] При сбросе координаты пересчитываются относительно глобального (0,0)
- [ ] Фигуры визуально не двигаются ни при создании, ни при сбросе маркера
- [ ] Новые фигуры создаются в координатах активного origin
- [ ] Toggle `A` — показать/скрыть оси X/Y от активного origin
