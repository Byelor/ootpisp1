# Глобальная точка отсчёта (World Origin)

---

## Концепция

На холсте всегда присутствует **один перемещаемый маркер (0,0)** — глобальная точка отсчёта. Все координаты всех фигур в панели свойств отображаются относительно этого маркера. Маркер можно свободно перетащить в любое место холста мышью — фигуры при этом визуально остаются на месте, но их координаты пересчитываются.

Аналог — **UCS (User Coordinate System)** в AutoCAD.

---

## Визуальное отображение

Маркер всегда виден на холсте, не исчезает при зуме и панорамировании:

```
      │
──────⊕──────   крестик с кружком
      │
    (0, 0)
```

- Цвет по умолчанию: красный (как оси в AutoCAD)
- При hover — маркер подсвечивается, курсор меняется на руку
- Опционально: от маркера через весь холст идут оси X (→) и Y (↑), toggle клавишей `A`

---

## Перемещение маркера

Маркер перетаскивается мышью напрямую — никаких специальных инструментов или кнопок:

```
Hover над маркером:
  курсор → sf::Cursor::Hand

MouseButtonPressed (Left) на маркере:
  state = DRAGGING_ORIGIN
  dragStartWorld = screenToWorld(mousePos)

MouseMoved (while DRAGGING_ORIGIN):
  newWorldPos = screenToWorld(mousePos)
  worldOrigin = newWorldPos
  // Фигуры визуально стоят на месте —
  // координаты в панели свойств пересчитываются автоматически

MouseButtonReleased:
  state = IDLE
```

**Важно:** фигуры не двигаются. Меняется только система отсчёта — как если бы ты переставил линейку на столе.

---

## Пересчёт координат фигур

Когда маркер перемещается на delta в мировых координатах:

```cpp
void Scene::moveWorldOrigin(sf::Vector2f delta) {
    for (auto& figure : figures) {
        figure->anchor -= delta;
        // Относительные координаты вершин не меняются —
        // меняется только anchor (абсолютная позиция)
    }
    worldOriginPos += delta;
}
```

Панель свойств обновляется в реальном времени пока тащишь маркер.

---

## Сброс на стандартное место

Стандартное место — **центр экрана** при запуске приложения (фиксируется один раз).

Способы сброса:

| Способ | Действие |
|---|---|
| Двойной клик по маркеру | Сброс origin в стандартное место |
| Кнопка в панели свойств `[Сбросить]` | То же самое |

При сбросе координаты всех фигур пересчитываются обратно относительно стандартной точки.

```cpp
void Scene::resetWorldOrigin() {
    sf::Vector2f delta = worldOriginPos - defaultOriginPos;
    moveWorldOrigin(-delta); // пересчитать фигуры обратно
    worldOriginPos = defaultOriginPos;
}
```

## В панели свойств

Отдельный блок в нижней части панели:

```
┌──────────────────────────────────┐
│  ТОЧКА ОТСЧЁТА                  │
│  X: [ 412.0 ]  Y: [ 308.0 ]     │  ← текущее положение маркера
│                  [Сбросить]      │
└──────────────────────────────────┘
```

- Поля X/Y — только для чтения (информационные), маркер двигается только мышью
- Кнопка `[Сбросить]` — возврат в стандартное место

---

## Взаимодействие с другими системами

| Система | Поведение |
|---|---|
| Зум | Маркер масштабируется вместе с холстом, всегда в правильном месте |
| Pan (Space + ЛКМ) | При сдвиге холста маркер двигается вместе с ним — он часть мирового пространства |
| Панель свойств | Все координаты фигур пересчитываются при каждом движении маркера |
| Bounding box | Пересчитывается автоматически вместе с координатами |

## Чеклист

- [ ] Маркер ⊕ постоянно виден на холсте
- [ ] Hover — курсор меняется на руку
- [ ] Drag мышью — маркер перемещается, фигуры стоят на месте
- [ ] Координаты всех фигур пересчитываются в реальном времени при drag
- [ ] Двойной клик по маркеру — сброс в стандартное место (центр экрана при запуске)
- [ ] Кнопка `[Сбросить]` в панели свойств — то же самое
- [ ] Текущие координаты маркера отображаются в панели свойств
- [ ] Toggle `A` — показать/скрыть оси X/Y от маркера
- [ ] Маркер корректно ведёт себя при зуме и панорамировании
